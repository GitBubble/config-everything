##  STEP2: Create Project  
- enter your project directory  
- bash import_symbol.sh
- start vim

##  HELP
 
- ctrl+n   : toggle nerdtree    
- enter    : toggle symbol list  
- cs find <tab> : find file or symbol in source  
   -- cs find s : find symbol    
   -- cs find g : find global     
   -- cs find f : find file
